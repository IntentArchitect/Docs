# Fundamentals #9: Managing and Updating Architectures

Learn how to manage and update application architecture, modules, and releases effectively during active development.{: .lead }

<div class="video-16x9"><iframe name="lessonVideo" src="https://www.youtube.com/embed/PbmhunDwbR8?rel=0" title="Video" allowfullscreen></iframe></div>

## Summary

This video walks you through managing and updating your application's architecture, demonstrating how to install modules, like the value objects pattern. Gareth also highlights the importance of making architectural changes only during active development phases, ensuring everything is checked into version control.
