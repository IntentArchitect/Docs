# Introduction to Intent Architect

Intent Architect is the first architecture-centric AI code automation platform for .NET developers.

It's the platform that delivers AI-driven development speed, with built-in guardrails and uncompromised quality, at any scale.

It brings design and architectural visibility to your software development processes and unlocks a single cohesive workflow to go from specification to production-ready code up to 10X faster. Architectural adherence enforced by default. Technical and cognitive debt minimized by design.

## Key Concepts

- [Visual Design Tools](xref:key-concepts.visual-modeling): Intent Architect's visual design tools let you create living blueprints of your system that realize directly as code, creating a powerful context engine to minimize technical and cognitive debt by design.
- [Architecture Enforcement](xref:key-concepts.deterministic-codegen): Guaranteed consistency and standardization by default. Deterministic architecture enforcement systems provide strong guardrails, ensuring consistent implementation of your design at any scale, with zero drift.
- [AI Agents](xref:key-concepts.non-deterministic-codegen): Intent Architect's customizable context engineering systems ensure your agents execute accurately and in full conformance with your design and architecture, out-the-box.
- [Codebase Control](xref:key-concepts.codebase-integration): Intent Architect works in an integrated way with your existing IDE, continuously realizing your design as code through a transparent review process. Developers stay in full control with complete visibility and freedom to customize any generated code.
